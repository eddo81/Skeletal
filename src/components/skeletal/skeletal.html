Skeletal

A bare bones front end starter kit

Hi!

If you're a nit-picky developer like me setting up a proper front end tool chain can prove to be a rather time consuming endeavor. In order to get up and running quickly with my own projects I've put together this little starter kit. Skeletal is a curation of Gulp tasks combined with Browserify and some nifty NPM scripts for good measure. Feel free to try it out for yourself.
